# 大学生就业分析可视化项目

## 项目概述

本项目是一个基于 Vue 3 的大学生就业分析可视化平台，旨在通过多维度数据分析，揭示2004-2024年间大学生就业市场的演变规律，帮助政策制定者、教育机构和学生更好地理解就业市场的动态变化。

## 功能特性

### 📊 核心功能

1. **故事模式**
   - 以时间线形式展示大学生就业市场的时代变迁
   - 从互联网萌芽期到智能时代的完整历程
   - 动态背景图片随时代切换

2. **探索模式**
   - 就业形势演变分析（时间序列图）
   - 专业结构演变分析（河流图）
   - AI技术影响分析（气泡矩阵图）
   - 行业流向变迁分析（桑基图）
   - 就业指导建议模块

3. **交互功能**
   - 学历筛选（本科/研究生）
   - 地区筛选（东部/中部/西部）
   - 年份时间轴选择
   - 专业详情卡片展示
   - 鼠标悬浮数据提示

## 技术栈

- **框架**: Vue 3 (Composition API)
- **数据可视化**: D3.js
- **桑基图**: d3-sankey
- **样式**: Tailwind CSS 3
- **构建工具**: Vite
- **语言**: JavaScript/TypeScript

## 使用说明

### 故事模式

首次进入网站默认进入故事模式，通过滚动页面浏览不同时代的就业特征：

1. **互联网萌芽期 (2004-2008)** - 传统行业主导
2. **移动互联网期 (2009-2014)** - IT行业崛起
3. **AI探索期 (2015-2019)** - 人工智能起步
4. **智能时代 (2020-2024)** - AI深度融合

### 探索模式

点击导航栏"探索数据"按钮切换到探索模式，可进行：

1. **筛选数据** - 通过学历和地区下拉框筛选
2. **查看图表** - 四个主要分析图表
3. **专业详情** - 点击专业查看详细信息和就业建议
4. **时间轴交互** - 使用滑块选择年份

### 图表说明

| 图表 | 类型 | 说明 |
|------|------|------|
| 时代总览 | 时间序列 | 就业率和薪资趋势 |
| 专业结构演变 | 河流图 | 各专业占比变化 |
| AI影响矩阵 | 气泡图 | 替代风险与发展潜力 |
| 行业流向变迁 | 桑基图 | 专业-行业就业流向 |

## 数据说明

项目使用模拟数据，涵盖：

- **时间范围**: 2004-2024年
- **学历层次**: 本科、研究生
- **地区划分**: 东部、中部、西部
- **专业类别**: 工学、理学、文学、经济学、管理学、医学、法学、教育学
- **就业行业**: IT/互联网、金融、教育、制造业、咨询等



## 自定义配置

### 添加新数据

在 `src/utils/dataLoader.js` 中添加或修改数据生成逻辑。

### 修改主题颜色

在 `src/style.css` 中调整 Tailwind 主题配置。

### 添加新图表

在 `src/components/` 目录创建新组件，并在 `App.vue` 中引入。
